# 航道断面算量自动化平台 v3.0

## 使用说明

1. 双击 `航道断面算量自动化平台v3.0@黄秉俊.exe` 启动前端
2. 前端会自动检测后端服务状态
3. 如需修改后端逻辑，可直接编辑 `server.py` 或 `engine_cad.py`

## 文件说明

- `航道断面算量自动化平台v3.0@黄秉俊.exe` - 前端启动器
- `server.py` - 后端服务脚本（可编辑）
- `engine_cad.py` 或 `engine_cad.pyd` - 核心算法模块
- `_internal/` - Python运行时依赖（请勿删除）

## 开发者说明

- 后端支持 `.py` 源码模式和 `.pyd` 编译模式
- 如果同时存在 `.py` 和 `.pyd`，优先使用 `.pyd`
- 修改 `server.py` 后重启平台即可生效
